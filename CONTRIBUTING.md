# Contributing to OSIRIS Review Room

## Getting started
1. Fork the repo
2. Run `npm install && npm run dev`
3. Open `http://localhost:3000`

## Where to make changes
- **Client view (client portal):** `src/components/ClientView.tsx`
- **Agency view (internal dashboard):** `src/components/InternalView.tsx`
- **Data model & API:** `server.ts`
- **Styling:** `src/index.css` (theme), Tailwind classes inline

## Pull request checklist
- Test your changes in both dark and light mode
- Make sure the build passes (`npm run build`)
- Update screenshots if you change the UI

## Need help?
Open an issue or reach out to `info@theosirislabs.com`.
